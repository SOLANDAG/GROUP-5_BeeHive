# BeeHive

BeeHive is an AI-assisted mobile service platform built with Expo and Firebase.  
The application connects customers with service providers and uses an intelligent assistant called **Bee** to help users find services, manage bookings, and schedule appointments.

BeeHive focuses on intelligent scheduling, service discovery, and real-time booking management.

---

# Project Concept

The BeeHive platform allows users to request services and connect with workers who provide those services.  
The system supports three main user roles:

Customer  
Service Provider (Worker)  
Administrator  

Customers can request services, while service providers offer services and manage their schedules.  
Administrators oversee the system and manage platform activity.

The platform uses an AI assistant named **Bee** to help users find services and manage scheduling automatically.

---

# Main System Modules

Customer Module
- Create account and login
- Search for services
- Book service providers
- Track booking status
- Chat with service providers
- View schedule and booking history

Service Provider Module
- Apply as a service provider
- Provide business and service details
- Manage availability
- Accept or decline bookings
- View ratings and service history

Admin Module
- Review provider applications
- Approve or reject workers
- Monitor platform activity
- Handle reports

---

# Bee AI Assistant

Bee is the intelligent assistant of the platform.

Bee helps users by:
- Finding available service providers
- Recommending services
- Managing booking schedules
- Providing help and app guidance

Bee also supports intelligent scheduling to help match customers with available workers.

# Installation

Clone the repository


git clone https://github.com/SOLANDAG/GROUP-5_BeeHive.git


Go to the project directory


cd BeeHive


Install dependencies


npm install


Start the Expo development server


npx expo start
