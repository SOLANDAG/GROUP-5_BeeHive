# BeeHive

BeeHive is an AI-assisted mobile application built using Expo and Firebase that connects customers with service providers. The platform allows users to discover services, submit booking requests, manage schedules, and communicate through a smart in-app assistant.

The system is designed to simplify service booking and task management through a modern mobile interface and an AI-powered assistant called Bee.

---

# Project Overview

BeeHive provides a centralized platform where users can:

• Find service providers  
• Book services and manage appointments  
• Track booking status in real time  
• Communicate through chat  
• Use an AI assistant to manage tasks  

The application supports both **customers and service providers**, allowing each role to manage their own activities within the system.

---

# Main Features

Customer Module
- Register and create an account
- Login using authentication
- Submit booking requests
- View booking history
- Track booking status (Pending, Accepted, Completed, Cancelled)

Provider Module
- Apply to become a service provider
- Define services and pricing
- Set available schedule
- Receive and manage booking requests

Admin Module
- Review provider applications
- Approve or reject providers
- Monitor platform activity

AI Assistant (Bee)
- Helps users find providers
- Helps create bookings
- Displays schedules
- Assists with app navigation

# Installation

Clone the repository
https://github.com/SOLANDAG/GROUP-5_BeeHive.git


Navigate to the project folder

cd BeeHive

Install dependencies

npm install

Start the development server

npx expo start

---

# Running the App

Once Expo starts, you can run the application in several ways:

Android Emulator  
iOS Simulator  
Expo Go mobile application  

You can also scan the QR code generated by Expo to run the app on a physical device.
